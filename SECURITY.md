# Security Policy

## Supported scope

Security fixes are applied to actively maintained packages and branches in this repository.

## Reporting a vulnerability

Please do not disclose vulnerabilities publicly before a fix is available.

Share a private report including:

- affected area or package
- reproduction steps or proof of concept
- impact assessment
- optional remediation suggestions

Maintainers will acknowledge the report, validate the issue, and coordinate a fix
before public disclosure.
